Titre du projet : Pipeline ETL & Modèle de prédiction du churn pour les clients e-commerce

Description :

Ce projet met en place un pipeline ETL pour transformer les données brutes d'e-commerce en couches Bronze, Silver et Gold, puis entraîne un modèle de Random Forest pour prédire le churn des clients à partir des métriques RFM.

Objectifs:

Nettoyer et transformer les données brutes (CSV → Parquet).
Créer des métriques RFM (recency, frequency, monetary) pour chaque client.
Identifier les clients churn (inactifs depuis un certain nombre de jours).
Entraîner un modèle de Random Forest pour prédire le churn.
Sauvegarder le modèle et le scaler pour usage futur.
